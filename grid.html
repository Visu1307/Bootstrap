The grid system is responsive, and the columns will re-arrange automatically depending on the screen size.

Grid Classes

The Bootstrap 5 grid system has six classes:

  • .col- (extra small devices - screen width < 576px) col-12
  • .col-sm- (small devices - screen width ≥ 576px) col-sm-10
  • .col-md- (medium devices - screen width ≥ 768px) col-md-8
  • .col-lg- (large devices - screen width ≥ 992px) col-lg-6
  • .col-xl- (xlarge devices - screen width ≥ 1200px) col-xl-4
  • .col-xxl- (xxlarge devices - screen width ≥ 1400px) col-xxl-2
