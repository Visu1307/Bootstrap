Sizing

Set heights using classes like .form-control-lg and .form-control-sm.

Disabled

Add the disabled boolean attribute on an input to give it a grayed out appearance and remove pointer events.

Readonly plain text

If you want to have <input readonly> elements in your form styled as plain text, use the .form-control-plaintext class to remove the default form field styling and preserve the correct margin and padding.

Color

Switches

A switch has the markup of a custom checkbox but uses the.form-check and .form-switch class to render a toggle switch. Switches also support the disabled attribute.

Default (stacked)

By default, any number of checkboxes and radios that are immediate sibling will be vertically stacked and appropriately spaced with .form-check.

Inline

Group checkboxes or radios on the same horizontal row by adding .form-check-inline after any .form-check.

Input Group

Place one add-on or button on either side of an input. You may also place one on both sides of an input. Remember to place <label>s outside the input group.
